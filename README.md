# Contribution

1. Clone Repo locally [`git clone`]
   - `git@github.com:dbo-keeganpatton/eyelady.git` for SSH
   - `https://github.com/dbo-keeganpatton/eyelady.git` for http
2. Create feature branch `git checkout -b [your-branch]`
3. Add code changes locally. - `git add .` - `git commit -m 'here is what I did'`
4. Sync with main branch before pushing back to remote.
   - `git pull origin main`
5. Push to this repo with `git push -u origin [your-branch]`
6. Open a Pull request from your branch to main
